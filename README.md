## **ğŸ“ Deskripsi Produk**

**LAPOR.SOLO â€” *Layanan Pengaduan Kota Solo***

Aplikasi pelaporan masyarakat berbasis mobile yang memudahkan warga Kota Surakarta dalam mengirimkan keluhan seputar Kota Solo secara cepat, jelas, dan terintegrasi.

- Mendukung kebijakan Wali Kota Solo dalam pengelolaan keluhan masyarakat.
- Mempermudah warga menyampaikan laporan dengan lokasi otomatis.
- Meningkatkan efektivitas pemerintah dalam menindaklanjuti laporan.
- Menyediakan visualisasi laporan melalui foto dan peta interaktif.

---

## **ğŸ§± Komponen Pembangun**

### **Tabel Komponen Utama**

| Komponen | Teknologi | Fungsi |
|---------|-----------|--------|
| **Frontend** | Ionic Framework, Angular | Membuat antarmuka aplikasi mobile. |
| **Backend** | Firebase Realtime Database | Menyimpan dan mengelola data laporan. |
| **Penyimpanan Foto** | Cloudinary | Menyimpan dan mengoptimasi foto laporan. |
| **Peta & Visualisasi** | Leaflet.js, OpenStreetMap | Menampilkan laporan di peta secara interaktif. |

---

## **ğŸ“š Sumber Data**

Aplikasi **LAPOR.SOLO** menggunakan beberapa sumber data utama untuk memastikan data laporan akurat dan dapat diakses secara real-time.

### **Tabel Sumber Data**

| Sumber Data | Digunakan Untuk | Detail Informasi |
|-------------|------------------|------------------|
| **Firebase Realtime Database** | Menyimpan data laporan | Kategori, keterangan, koordinat, timestamp |
| **Cloudinary** | Penyimpanan media (foto) | Optimasi foto, URL akses cepat |
| **GPS / Geolocation Perangkat** | Mendapatkan lokasi otomatis | Sensor GPS perangkat dan Geolocation API |
| **OpenStreetMap + Leaflet** | Basemap & visualisasi | Menampilkan peta dasar dan titik laporan |
